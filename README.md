## Descrição

Projeto teste desenvolvido em Nodejs + Express + Nestjs Framework + Postgres

## Setup 

```bash
$ npm install
```

## Compilar e rodar 

```bash
# development
$ npm run start

# debug
$ npm run start:dev

# production mode
$ npm run start:prod
```

# Docker
### Para rodar localmente com docker

```bash 
$ docker compose up -d
```
