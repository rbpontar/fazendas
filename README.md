## Descrição

Projeto teste desenvolvido em Nodejs + Express + Nestjs Framework + Postgres

## Setup 

```bash
$ npm install
```

## Compilar e rodar 

```bash
# development
$ npm run start

# debug
$ npm run start:dev

# production mode
$ npm run start:prod
```

# Docker
### Para rodar localmente com docker

```bash 
$ docker compose up -d
```

# Swagger - Documentação
### Para testes locais com swagger

```bash 
$ http://localhost:3000/api
```
